Introduction¶

In this work time series model learn with incoming data and predict next 7 days prediction of Daily confirmed, Daily deceased, Daily recovered, Total confirmed, Total deceased, Total recovered. Model is manually calibrated by inspection method. As adrupt change in data in last few days observed, so, all 63 data points are taken of training the model. Whatever data will come after $3^{nd}$April 2020 will be used to test the model.

Time series forcasing is an application of supervised training. A model is build to predict values from date with past data as training data by feeding the model with future dates, it produces forcasted values for the future dates.

Model is build by finding a best smooth line which can be represented as sum of the components like overall growth trends, yearly seasonality, weekly seasonality, Holiday effects.

Prophet time series = Trend + Seasonality + Holiday + error

Trend models non periodic changes in the value of the time series.

Seasonality is the periodic changes like daily, weekly, or yearly seasonality.

Holiday effect which occur on irregular schedules over a day or a period of days.

Error terms is what is not explained by the model.

The advantage of the this model over other Time series model (like ARIMA,SARIMA etc.) are

  • Work best on the strong seasonal effect data as it support yearly, weekly, and daily seasonality, plus holiday effects
  • It handels outliers as well
  • It is robost in handeling missing values

#### Forecaster model
def forecast(df,ds,y,y_label='y',title="Projection", plot=False):
    df = df[[ds, y]]
    df=df.rename(columns={ds: 'ds', y: 'y'})
    prophet = fbprophet.Prophet(changepoint_range=0.9,changepoint_prior_scale=0.2,weekly_seasonality=False,daily_seasonality=True,yearly_seasonality=False)
    prophet.fit(df)
    forecast = prophet.make_future_dataframe(periods=forcast, freq='D')# make_future_dataframe() to which we specify the number of days to extend into the future. 
    forecast = prophet.predict(forecast)
    # Avoid negative
    forecast["yhat"] = np.where(forecast["yhat"]<0,0,forecast["yhat"])
    forecast["yhat_lower"] = np.where(forecast["yhat_lower"]<0,0,forecast["yhat_lower"])
    forecast["yhat_upper"] = np.where(forecast["yhat_upper"]<0,0,forecast["yhat_upper"])
    if plot :
        fig=prophet.plot(forecast, xlabel = 'Date', ylabel = y_label)
        a = add_changepoints_to_plot(fig.gca(), prophet, forecast)
        plt.title(title);
    return (forecast,df)

# Nash-Sutcliffe Efficiency
def nse(simulation_s, evaluation):
    nse_ = 1 - (np.sum((evaluation - simulation_s) ** 2, axis=0, dtype=np.float64) /
                np.sum((evaluation - np.mean(evaluation)) ** 2, dtype=np.float64))
    return nse_
# Root Mean Square Error
def rmse(simulation_s, evaluation):
    rmse_ = np.sqrt(np.mean((evaluation - simulation_s) ** 2, axis=0, dtype=np.float64))
    return rmse_
